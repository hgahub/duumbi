# Phase 15: Studio Workflow Redesign — GitHub Issues

## Milestone: Phase 15 — Studio Workflow Redesign

**Description:** Streamline the Duumbi Studio from an 8-step workflow to a 3-panel design (Intents → Graph → Build). Connect Studio chat to real LLM, add 3 sample end-to-end tasks as kill criterion.

**Kill Criterion:**
1. New user completes Calculator sample in CLI REPL in <10 minutes
2. All 3 sample tasks (Calculator, String Utils, Math Library) succeed in Studio
3. Studio chat sends real LLM requests and displays streaming responses

### Issue #501: WebSocket chat endpoint for Studio
**Labels:** `enhancement`, `studio`, `phase-15`

Add `/ws/chat` WebSocket endpoint to the Studio axum server.

**Acceptance Criteria:**
- [ ] WebSocket upgrade at `/ws/chat` route in `crates/duumbi-studio/src/lib.rs`
